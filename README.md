### Description
- Shapes API for ASB.
-  Built using [Nest](https://github.com/nestjs/nest) framework.
<br />

## Installation
```bash
$ npm install
```

## Post Installation
>##### ensure you have postgres installed and the database created.
<br />

### create tables in database
```bash
$ npm run migration
```
<br />

## Running the app
<br />

### development
```bash
$ npm run start
```

### watch mode
```bash
$ npm run dev
```

### production mode
```bash
$ npm run prod
```

## Test
### unit tests
```bash
$ npm run test
```

### e2e tests
```bash
$ npm run test:e2e
```

### test coverage
```bash
$ npm run test:cov
```
